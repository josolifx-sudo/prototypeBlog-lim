# MEVN Blog App (StringsBlog)

## Tech Stack
Backend: Node, Express, MongoDB, Mongoose, JWT, bcrypt
Frontend: Vue 3, Vite, Vue Router, Pinia, Axios

## Features
- Register and Login with hashed password
- JWT authentication
- Public feed: view all posts
- Public post view: view one post and its comments
- Auth users: create posts, comment on posts
- Ownership authorization: users can edit and delete their own posts
- Admin: can delete any post and remove any comment
